# BUG-003 - Checklist Data Field Mismatch

## Status
**fixed** ✅

## Severity
HIGH

## Description
El component ChecklistComponent buscava dos camps diferents (`items` i `checklist`) sense consistència, causant confusió en el data model.

## Location
- **File:** `src/components/card/ChecklistComponent.tsx`
- **Line:** 24

## Original Code
```tsx
const checklist = data?.items || data?.checklist || []
```

## Fix Applied
1. Estandarditzar al camp `items` com a únic camp per checklist
2. Actualitzar la interfície per reflectir només `items`

## Changes Made
```tsx
// Line 10-13: Interface updated
interface ChecklistComponentProps {
  data: {
    items?: ChecklistItem[]
  }
  ...
}

// Line 24: Simplified to use only 'items'
const checklist = data?.items || []
```
