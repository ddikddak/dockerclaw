# BUG-008: JSON Parsing Error Handling

**Status:** fixed  
**Priority:** MEDIUM  
**Assigned to:** Nestor  
**Reported by:** Albert

## Problem
Quan el request body té JSON malformat, retorna 500 en lloc de 400.

## Root Cause
Faltava handler específic per `SyntaxError` de `express.json()` abans del handler d'errors genèric.

## Fix
- Afegir check per `SyntaxError` amb propietat `body` al error handler
- Retornar 400 Bad Request amb `{ error: "Invalid JSON" }`

- `81f4846` fix(api): BUG-008 handle JSON parsing errors as 400
