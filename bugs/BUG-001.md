## Descripció
Els botons approve/reject/delete/archive a les cards no es mostren perquè els handlers (`onApprove`, `onReject`, etc.) no es passen des dels components pare (Board → Column → Card).

## Problema
- Handlers no wirejats al component Card
- Inline editing també trencat (`onEditText`, `onToggleCheck`)

## Impacte
Funcionalitat core trencada - humans no poden interactuar amb cards.

## Fix Proposat
1. Afegir handler functions a Board.tsx
2. Passar handlers a través de Column fins a Card
3. Testejar cada acció

## Fitxers Afectats
- `src/components/Board.tsx`
- `src/components/Column.tsx`
- `src/components/Card.tsx`
