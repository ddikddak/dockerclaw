## Descripció
Tots els API routes a `frontend/src/app/api/` retornen 404 en mode desenvolupament.

## Problema
- Build OK, però runtime falla
- HTTP requests retornen HTML 404 en comptes de JSON

## Endpoints Afectats
- `POST /api/agents/register`
- `GET /api/agents/:id/events`
- `POST /api/cards/:id/actions`
- `POST /api/cards/:id/components/:cid/actions`

## Impacte
API layer completament trencat. Cap testing possible.

## Root Cause
La configuració `turbopack.root` a `next.config.ts` interferia amb App Router.

## Fix Aplicat
Eliminada la configuració `turbopack.root` de `next.config.ts`.

## Verificació
- [x] Build passa correctament
- [x] Totes les rutes API apareixen a la llista de rutes
- [x] No hi ha errors de compilació
